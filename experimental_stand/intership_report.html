Intership Report

Goal: To develop a concept of an experimental stand, which are:

  1. Portable (can be transported and installed on any ground). Installation pillars should have a possibility to be fixed on the ground.
  1. Different legs can be installed. Even more, electricity should be provided for the base of a leg
  1. The base of a stand should have a container, where dirt, grass, etc can be installed. A strong guy should pick it up.
  1. The possibility to install extra load on a leg shaft
  1. The leg should have a possibility to turn on a shaft and move up and down (can be done by linear guideway)
  1. A place for install microcomputer (Jetson TX2 or Nano), batteries, some other sensors

The device should be as small as possible without losing functionality. Any improvements are welcome.

Precise:

  1. Container for dirt, dull
  1. Base foot which can be fix on a container
  1. Core part, consist of bearings, slip rings, linear guideway
  1. Top of a stand, which has a place for batteries, microcomputer etc.
  1. Installation pillars.

Numbers:

  1. Load on the leg is 300 N at max
  1. Leg size - 50-200 mm

A brief description of achieved results

A modular stand that satisfies the requirements set by the instructor.

The Design itself:

  1. The Base

Elevated base for ease of cleaning, I intended the design to be modular so that it can be clean, disasemble, and reasemble easily. 3 gaps in the holding area for ease of cleaning.

The base consist of the shield and the support, inspired by wood working techniques, I designed them with interlocking parts like lego, that can be assembled without any screws or hinges.

2. The Lid:

The lid has enough area to install batery pack and electronics with holes to pass slip rings through.

3. Minor details:

Weight holder and 8mm rod holder with double bearings inside.
